# riscv-sha256-accelerator
• Reduced cryptographic execution hardware-software on a 32-bit Pulpino RISC-V SoC.  • Designed a 64-round parallelised FSM and 8-stage pipelined adder tree for SHA-256; integrated as APB-mapped slave  IP on Altera Cyclone IV FPGA at 50 MHz.  • Validated timing closure with 46.3% avalanche effect; work published in  journal (Jan–Apr 2026).
